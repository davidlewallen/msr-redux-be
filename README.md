# msr-redux-be

## Prereqs

- Install MongoDB locally [Download Here](https://www.mongodb.com/download-center/community)

## Setup

- `git clone https://github.com/davidlewallen/msr-redux-be.git`
- `cd msr-redux-be`
- Create a `.env` file and get keys from maintainer
- `npm install`
- `npm run dev`
- http://localhost:8000
